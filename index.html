Welcome to My Web Project

This project showcases how HTML, CSS, and JavaScript can be combined to create an interactive, responsive, and user-friendly web application.

Explore the sections to learn about form building, validation, responsive layouts, and dynamic DOM manipulation using a to-do list example.

About This Project

The goal of this project is to enhance web development skills through hands-on practice. It follows four steps:

  • 1. Contact Form: Built with HTML and CSS for structured data collection.
  • 2. Form Validation: Implemented with JavaScript to ensure accurate inputs.
  • 3. Responsive Layout: Designed using Flexbox, CSS Grid, and media queries.
  • 4. To-Do List: A dynamic example of DOM manipulation with JavaScript.

Together, these steps demonstrate the foundation of creating professional websites.
